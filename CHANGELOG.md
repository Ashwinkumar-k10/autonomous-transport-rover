## [v1.0.0] — 2026-02-20

### 🏆 Competition Release — Pragyan NIT Trichy, Ingenium'26

### Added
- ESP32-S3 firmware with PlatformIO build system
- 5-channel IR array line following with PID control
- RFID RC522 junction recognition and station docking logic
- Poor Man's LIDAR: servo-swept HC-SR04 obstacle scanner (30°–150°)
- HX711 load cell payload monitoring with overload detection
- MPU6050 IMU tilt safety cutoff
- Mechanical bump sensor collision detection
- BTS7960 dual motor driver control
- Real-time WebSocket web dashboard
- Dashboard panels: IR status, obstacle distance, RFID log, payload, IMU, alerts
- Full integration test: 12/12 test cases passed
